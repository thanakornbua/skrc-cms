Text input with an uppercase mono label and a violet focus glow. Use `mono` for any technical value (student ID, registration number, code).

```jsx
<Input label="ชื่อ-นามสกุล / Full name" placeholder="กรอกชื่อ" />
<Input label="เลขประจำตัวสอบ / Student ID" mono placeholder="SKRC-2026-____" />
```

surface-2 fill, radius-sm. Focus turns the border violet and adds a soft purple glow. Pair with the primary `Button` for submit.
