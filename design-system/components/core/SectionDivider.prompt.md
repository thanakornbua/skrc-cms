Full-bleed gradient band that separates sections on slides and web pages. Thai label on top, English below.

```jsx
<SectionDivider thai="ส่วนที่ 2 — เซนเซอร์" en="Part 2 — Sensors" />
<SectionDivider thai="ผลการคัดเลือก" en="Selection Results" height={48} />
```

Use `height={80}` for slides, `height={48}` for web. Always uppercase mono, white text, gradient fixed at 135deg.
