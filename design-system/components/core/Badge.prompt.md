Pill badge / tag for category labels and pass-fail status. Uppercase IBM Plex Mono, text-xs.

```jsx
<Badge>Sensors</Badge>
<Badge variant="secondary">Optional</Badge>
<Badge variant="success">ผ่าน</Badge>
<Badge variant="error">ไม่ผ่าน</Badge>
```

Variants: `primary` (gradient fill, white) for category tags; `secondary` (surface-2 + violet) for secondary tags; `success` / `error` (15%-tint fill + semantic text) for status. Keep labels to 1–2 words.
