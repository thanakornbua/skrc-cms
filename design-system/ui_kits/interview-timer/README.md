# Interview Timer — UI kit

Front-of-room dashboard for interview day. Shown on a projector while candidates are
called in turn.

- `index.html` — interactive entry point. The countdown ticks live.
- `InterviewTimer.jsx` — current-candidate card with the large gradient countdown
  (text-5xl mono), a next-up card, and the live queue (current row highlighted, past
  rows dimmed to 40%).
