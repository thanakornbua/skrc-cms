บทที่ 04 — PID CONTROL

Line-Following Logic

การควบคุมหุ่นยนต์เดินตามเส้นด้วย PID
  • อ่านค่าจากเซนเซอร์ตรวจจับเส้น
  • คำนวณค่าความคลาดเคลื่อน (error)
  • ปรับความเร็วมอเตอร์ซ้าย–ขวา
  • ทดสอบและจูนค่า Kp, Ki, Kd
# คำนวณค่าแก้ไข — compute correction
error = target - read_line()
turn  = Kp * error
drive(base + turn, base - turn)
