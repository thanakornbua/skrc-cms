RULES

Do & Don't

Do — Typography
  • Use IBM Plex Mono for all technical identifiers, code, labels, nav.
  • Use gradient text for headings & section labels.
Don't — Typography
  • Use decorative or serif fonts anywhere.
  • Use gradient text for body copy.
Do — Color
  • Use the gradient as an accent (dividers + button fills excepted).
  • Use semantic colors consistently for pass/fail.
Don't — Color
  • Use #000000 — use --color-dark #1a1a2e.
  • Use pure white page bg — use --color-bg #fdfbff.
Do — Spacing & Shadow
  • Use the 8px grid for every spacing decision.
  • Use the purple-tinted shadow variables.
Don't — Spacing & Shadow
  • Mix spacing values outside the scale.
  • Stack multiple shadows on one element.
Do — Gradient
  • Always 135deg, magenta → violet → blue in order.
Don't — Gradient
  • Rotate, reverse, or reorder the stops.
